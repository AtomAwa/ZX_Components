# 基础弹窗使用规范

组件说明：模态的基本对话弹窗

使用情景：需要中断用户当前页面操作去进行更多交互时使用

- a.内嵌表单
- b.分步操作
- c.导入文件
- d.复杂信息展示

## 全局规则
**弹窗大小**

【弹窗宽度】：最大不能超过宽度的 90%

【弹窗高度】：最小250px，最大为90%页面高度，一般在300-650px

**弹窗位置**

弹窗在页面内居中显示，弹窗外页面区域覆盖蒙版，背景滚动条锁定

**关闭规则**

除了关闭按钮和取消按钮，点击键盘的Esc也可关闭弹窗

**使用规则**

（除全屏弹窗外）只允许同时存在一个弹窗，若弹窗中继续出现二级功能页面，考虑刷新弹窗页面，气泡框等其他形式
